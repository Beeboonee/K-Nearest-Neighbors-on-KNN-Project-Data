# K-Nearest-Neighbors-on-KNN-Project-Data

This project applies the K Nearest Neighbors algorithm using the KNN Project Data dataset to classify the target class.

## Project Overview
The notebook follows the main machine learning classification steps:
- Importing the required libraries
- Uploading and reading the dataset
- Exploring the data
- Visualizing feature relationships
- Standardizing the feature values
- Splitting the data into training and testing sets
- Training the KNN model
- Making predictions
- Evaluating the model using a confusion matrix and classification report
- Testing multiple K values to find a better model performance

## Dataset
The dataset used in this project is:
- `KNN_Project_Data`

## Files
- `KNN_Project_Data`
- `KNN_Project_Assignment.ipynb`

- Python

## Objective
The objective of this project is to build a KNN classification model and evaluate its performance before and after selecting a better K value.

## Output
The final output includes:
- Data exploration
- Feature standardization

- Predictions on test data
- Confusion matrix
- Classification report
- Error rate vs K value plot
